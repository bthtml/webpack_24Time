# webpack_24Time
24小时网络科技-膜app

  # 目录说明     
  
 ## -src目录：生产目录   
 
 ### --assetse:公共目录

 ----component.js:vue组件   
 ----event.js:控制整个app的路由，事件推送，网络查询   

 ### ---pages：静态文件目录   
 ----html_template:模板文件，复制即可
